# 📊 Control Presupuestal y Análisis de Ahorros

Dashboard enfocado en el análisis financiero y control de presupuesto, permitiendo identificar desviaciones, ahorros y oportunidades de mejora.

---

## 🚀 Herramientas utilizadas
- Excel
- Tablas dinámicas
- Análisis financiero

---

## 📌 Objetivo
Comparar el presupuesto contra el gasto real para identificar desviaciones y evaluar el desempeño financiero de la organización.

---

## 📊 Análisis realizado
- Comparación mensual de presupuesto vs gasto real
- Identificación de ahorro y sobrecostos
- Análisis de variaciones anuales (2025 vs 2026)
- Seguimiento de costos totales
- Detección de meses con mayor impacto financiero

---

## 📈 Resultados
- Identificación de ahorro acumulado
- Detección de desviaciones presupuestarias
- Apoyo en toma de decisiones financieras
- Visualización clara del desempeño económico
